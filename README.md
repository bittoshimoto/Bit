What is Bit Core?
---------------------

Bit is a decentralized cryptocurrency based on the Bitcoin protocol.
Launched with the vision of offering a secure, fast, and scalable digital currency,
Bit aims to provide a transparent and community-driven alternative to traditional financial systems.
It uses the proven SHA-256 Proof of Work algorithm for consensus, ensuring both security and fairness for miners and users alike.
